# Dungeon Quest 3D

A lightweight browser-based 3D roguelike adventure built with `Three.js`. Explore procedurally generated dungeons, battle monsters, collect loot, and progressively upgrade your character as you descend deeper into challenging floors.

## Gameplay

Dungeon Quest 3D is a roguelike dungeon crawler where you navigate 3D procedurally generated floors filled with enemies and treasure. Defeat slimes, goblins, orcs, and a final boss to claim rewards and progress to the next floor. Each run is unique—find coins for upgrades, potions for healing, and permanent power-ups to increase your stats. Complete quests for extra rewards and challenge yourself to reach lower dungeons.

**Playable Classes:**
- **Tank** - High health, moderate attack speed, steady damage
- **Hunter** - Balanced stats with good damage output
- **Assassin** - Lower health but highest damage and speed
- **Rogue** - Flexible all-rounder with good survivability

## Features

- **Procedurally Generated Dungeons** - Every run is different with randomly placed enemies, chests, and loot
- **Diverse Enemies** - Face slimes, goblins, orcs, and a powerful boss with unique combat mechanics
- **Town Hub** - Return to a safe zone to access the shop, view stats, and prepare for the next dungeon
- **Resource Management** - Collect coins, heal potions, and permanent upgrades
- **Combat Mechanics** - Melee attacks, dash movement, potion use, and tactical positioning
- **Progression System** - Unlock power-ups, complete quests, and progress through increasingly difficult floors
- **Floor Rewards** - Earn coins and potions based on your performance in each dungeon
- **3D First-Person View** - Immersive dungeon exploration with dynamic lighting and shadows

## Run

Open `index.html` in a browser with internet access.

If your browser blocks local script loading, run a small local server from this folder:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.
